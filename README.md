This project implements a small data processing pipeline in a Linux environment to retrieve, validate, clean, data from the NASA Exoplanet Archive. 
The goal was to simulate a simplified version of a scientific data workflow, focusing on data quality, reproducibility, and structured processing.

☆ Objectives
- Practice working with real-world astronomical datasets  
- Perform structured data validation and cleaning  
- Build a reproducible data pipeline  
- Create visualisations of scientific data  
- Practice Linux-based workflows and automation

☆ Findings
- The dataset contained ~6158 entries, with ~6–7% removed during cleaning due to missing key parameters  
- No non-physical values were detected in the selected variables  
- Missing data is the primary limitation, reflecting observational constraints in astronomy  
- Log-scaled plots - necessary due to the wide dynamic range of physical values
